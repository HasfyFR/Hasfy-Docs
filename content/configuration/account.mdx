# Gestion du compte

La section **Mon compte** vous permet de gérer l'ensemble de vos informations personnelles et vos préférences de sécurité. Accédez-y depuis le menu utilisateur en haut à droite de l'interface, puis **« Mon profil »**, ou directement via **Paramètres &gt; Mon compte**.

## Informations du profil

### Modifier vos informations

Depuis l'onglet **Profil**, vous pouvez éditer :

- **Prénom et nom** : affichés dans l'interface, les assignations de tickets et les notifications. Saisissez votre nom complet tel que vous souhaitez qu'il apparaisse à vos collègues
- **Poste / Fonction** : votre intitulé de poste (ex. « Responsable IT », « Technicien support N2 »). Affiché sur votre profil public dans l'organisation
- **Numéro de téléphone** : utilisé pour les notifications SMS critiques (disponible sur les plans Pro et Enterprise)
- **Bio** *(optionnel)* : une courte description visible par les autres membres de votre organisation

Cliquez sur **« Enregistrer les modifications »** pour valider vos changements.

### Photo de profil (Avatar)

Pour ajouter ou modifier votre avatar :

1. Cliquez sur l'image de profil actuelle (ou l'icône par défaut)
2. Sélectionnez un fichier image depuis votre ordinateur
3. Formats acceptés : **JPG, PNG, WebP**
4. Taille maximale : **2 Mo**
5. L'image est automatiquement recadrée en format carré
6. Cliquez sur **« Enregistrer »**

Pour supprimer votre avatar et revenir à l'icône par défaut, cliquez sur **« Supprimer la photo »**.

## Changer d'adresse email

La modification de l'adresse email associée à votre compte nécessite une vérification en deux étapes :

1. Accédez à **Mon compte &gt; Sécurité &gt; Email**
2. Saisissez votre **nouvelle adresse email**
3. Confirmez avec votre **mot de passe actuel**
4. Cliquez sur **« Mettre à jour l'email »**
5. Un email de confirmation est envoyé à la **nouvelle** adresse
6. Cliquez sur le lien de confirmation dans cet email
7. Votre adresse email est mise à jour

> **Important** : Tant que vous n'avez pas confirmé la nouvelle adresse, votre ancienne adresse email reste active. Si vous ne recevez pas l'email de confirmation, vérifiez votre dossier spam ou cliquez sur **« Renvoyer le lien »**.

## Changer de mot de passe

Pour modifier votre mot de passe :

1. Accédez à **Mon compte &gt; Sécurité &gt; Mot de passe**
2. Saisissez votre **mot de passe actuel**
3. Saisissez votre **nouveau mot de passe** en respectant les critères :
   - Minimum 8 caractères
   - Au moins une lettre majuscule
   - Au moins une lettre minuscule
   - Au moins un chiffre
   - Caractères spéciaux recommandés
4. Confirmez le nouveau mot de passe
5. Cliquez sur **« Mettre à jour le mot de passe »**

Un indicateur de force du mot de passe vous aide à choisir un mot de passe robuste. Après modification, toutes vos sessions actives sont conservées mais vous pouvez les révoquer manuellement.

### Mot de passe oublié

Si vous avez oublié votre mot de passe :

1. Sur la page de connexion, cliquez sur **« Mot de passe oublié ? »**
2. Saisissez l'adresse email de votre compte
3. Un email contenant un lien de réinitialisation est envoyé
4. Cliquez sur le lien (valide 1 heure) et définissez un nouveau mot de passe

## Préférences de notification du compte

Gérez les notifications liées spécifiquement à votre compte :

| Type de notification | Description | Email | In-app |
|---------------------|-------------|-------|--------|
| Connexion depuis un nouvel appareil | Alerte si votre compte est utilisé depuis un appareil inconnu | ✅ | ✅ |
| Modification de mot de passe | Confirmation après changement de mot de passe | ✅ | — |
| Modification d'email | Confirmation de changement d'adresse | ✅ | — |
| Expiration de session | Rappel lorsque votre session est sur le point d'expirer | — | ✅ |

Pour modifier ces préférences, rendez-vous dans **Mon compte &gt; Notifications**.

## Sessions actives

La section **Sessions actives** affiche la liste de toutes les sessions ouvertes pour votre compte :

- **Appareil** : type d'appareil et navigateur utilisé
- **Localisation approximative** : basée sur l'adresse IP
- **Date de dernière activité**
- **Session actuelle** : marquée d'un badge « Actuelle »

Vous pouvez **révoquer** une session individuelle en cliquant sur **« Déconnecter »**, ou révoquer toutes les autres sessions en cliquant sur **« Déconnecter tous les appareils »**.

> **Conseil de sécurité** : Si vous repérez une session que vous ne reconnaissez pas, révoquez-la immédiatement et changez votre mot de passe.

## Suppression du compte

Pour supprimer définitivement votre compte Hasfy :

1. Accédez à **Mon compte &gt; Zone dangereuse**
2. Cliquez sur **« Supprimer mon compte »**
3. Une fenêtre de confirmation vous avertit des conséquences :
   - Suppression irréversible de toutes vos données personnelles
   - Retrait de toutes les organisations dont vous êtes membre
   - Si vous êtes **Owner**, vous devez transférer la propriété avant la suppression
4. Saisissez **« SUPPRIMER »** pour confirmer
5. Votre compte est supprimé sous 30 jours (période de grâce permettant l'annulation)

> **Attention** : La suppression du compte est irréversible après la période de grâce de 30 jours. Toutes les données associées seront définitivement effacées conformément au RGPD.
