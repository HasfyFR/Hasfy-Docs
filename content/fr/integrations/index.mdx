# Intégrations

Hasfy se connecte à votre écosystème IT pour centraliser la gestion de votre parc et automatiser vos workflows. Cette page présente ce qui est inclus nativement dans votre abonnement Hasfy Pro, les intégrations disponibles dès aujourd'hui dans la marketplace, et celles prévues sur notre feuille de route.

## Inclus dans Hasfy Pro

Deux capacités essentielles font partie du socle Hasfy Pro et ne sont **pas des intégrations à activer** : elles sont disponibles dès la création de votre organisation, sans configuration supplémentaire.

### Agent de supervision (monitoring)

Installé sur chaque équipement que vous souhaitez superviser, l'agent Hasfy collecte automatiquement les métriques système (CPU, RAM, disque, réseau, uptime) et les remonte vers votre tableau de bord.

➡️ [Installer l'agent Hasfy](/quickstart#étape-4--installer-lagent-hasfy)

### API REST

L'API REST de Hasfy est incluse dans l'abonnement Pro et accessible via des clés API, depuis **Paramètres &gt; API**.

➡️ [Documentation de l'API Hasfy](/api)

## Marketplace d'intégrations

Toutes les intégrations se gèrent depuis la page **Intégrations**, accessible depuis le menu de navigation principal (section **Automatisation**, icône puzzle 🧩). Cette page est réservée aux rôles **Owner** et **Admin**.

C'est une marketplace organisée par catégories, accessibles via des onglets :

Tout · Comptabilité · Communication · Sécurité · Supervision · CRM · Productivité · Paiement · Calendrier

![Page Intégrations : vue marketplace avec grille des catégories et statuts](/images/integration-page.jpg)

Une barre de recherche permet de filtrer les intégrations par nom. Chaque intégration affiche l'un des statuts suivants :

| Statut | Signification |
|--------|----------------|
| **Connecté** | L'intégration est active sur votre organisation |
| **Disponible** | Peut être activée immédiatement (bouton « Activer » / « Gérer ») |
| **En développement** | Annoncée et en cours de développement, pas encore utilisable |
| **Bientôt disponible** | Sur la feuille de route, date de disponibilité non communiquée |

## Disponible aujourd'hui

### Mollie Connect : Paiement

**Mollie Connect** est, à ce jour, la seule intégration activable depuis la marketplace. Elle vous permet de connecter votre propre compte Mollie pour encaisser vos clients :

- Création de **liens de paiement** sur vos factures clients
- **Suivi en temps réel** du statut des paiements
- **Émission de factures** directement depuis Hasfy
- Gestion de vos **profils de paiement** Mollie

> **À ne pas confondre** : Mollie Connect concerne l'encaissement de **vos clients**, via votre propre compte Mollie. Le paiement de **votre abonnement Hasfy Pro** utilise aussi Mollie, mais se gère séparément depuis [Paramètres &gt; Facturation](/configuration/billing).

➡️ [Configurer Mollie Connect](/integrations/mollie-connect)
