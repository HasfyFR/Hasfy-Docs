# Marketplace Techniciens

La **Marketplace Techniciens** de Hasfy met en relation les organisations ayant besoin d'interventions techniques avec un réseau de **techniciens qualifiés et vérifiés**. Que vous manquiez de ressources internes ou que vous ayez besoin d'une expertise spécifique, la marketplace vous permet de trouver le bon technicien rapidement.

> **Disponibilité** : La Marketplace Techniciens est disponible sur les plans **Pro** et **Enterprise**.

## Quand utiliser la marketplace

La marketplace est particulièrement utile dans les situations suivantes :

- **Surcharge temporaire** : votre équipe interne est débordée et vous avez besoin de renforts ponctuels
- **Expertise spécifique** : vous avez besoin d'un spécialiste dans un domaine que votre équipe ne couvre pas (réseau, sécurité, virtualisation, etc.)
- **Couverture géographique** : vous avez des équipements dans une zone éloignée de votre équipe
- **Intervention urgente** : un incident critique nécessite une intervention immédiate et votre équipe n'est pas disponible
- **Pas d'équipe IT interne** : vous êtes une PME sans service informatique dédié

## Rechercher un technicien

### Accès à la marketplace

1. Depuis le menu principal, accédez à **Marketplace &gt; Techniciens**
2. La page affiche la liste des techniciens disponibles dans votre zone géographique

### Filtres de recherche

Affinez votre recherche avec les filtres suivants :

| Filtre | Description |
|--------|-------------|
| **Localisation** | Ville, département ou rayon autour d'une adresse |
| **Spécialités** | Réseau, Systèmes, Sécurité, Matériel, Logiciel, Cloud, Impression, Téléphonie |
| **Disponibilité** | Disponible maintenant, cette semaine, date spécifique |
| **Type d'intervention** | Sur site, À distance |
| **Note minimale** | Filtrer par note moyenne (ex. 4 étoiles et plus) |
| **Certifications** | Microsoft, Cisco, CompTIA, ITIL, etc. |
| **Tarif** | Fourchette de tarif horaire |

### Profil d'un technicien

Chaque technicien dispose d'un profil détaillé affichant :

- **Informations générales** : nom, photo, localisation, zone d'intervention
- **Spécialités** : domaines d'expertise avec niveau (débutant, confirmé, expert)
- **Certifications** : certifications professionnelles vérifiées
- **Expérience** : années d'expérience et nombre d'interventions réalisées via Hasfy
- **Disponibilités** : créneaux disponibles pour les prochaines semaines
- **Tarifs** : tarif horaire indicatif (le tarif final est défini dans le devis)
- **Note moyenne** : évaluation sur 5 étoiles basée sur les avis des clients
- **Avis clients** : commentaires détaillés des précédentes interventions

## Demander un devis

### Processus de demande

1. Depuis le profil d'un technicien (ou depuis la liste), cliquez sur **« Demander un devis »**
2. Remplissez le formulaire de demande :
   - **Titre de l'intervention** : description courte du besoin
   - **Description détaillée** : expliquez le problème ou le travail à réaliser avec autant de détails que possible
   - **Type d'intervention** : sur site ou à distance
   - **Adresse d'intervention** *(si sur site)* : lieu exact de l'intervention
   - **Urgence** : Normal (sous 7 jours), Prioritaire (sous 48 h), Urgent (sous 24 h)
   - **Équipements concernés** : sélectionnez les équipements liés depuis votre inventaire
   - **Date souhaitée** *(optionnel)* : date et créneau préférés
   - **Pièces jointes** : photos, captures d'écran, documentations techniques
3. Cliquez sur **« Envoyer la demande »**

### Réponse du technicien

Le technicien reçoit votre demande et dispose de **48 heures** pour répondre avec :

- **Acceptation avec devis** : montant estimé, durée prévue, conditions
- **Demande de précisions** : questions complémentaires avant de pouvoir chiffrer
- **Refus** : le technicien n'est pas disponible ou ne couvre pas ce type de demande

### Validation du devis

Une fois le devis reçu, vous pouvez :

- **Accepter** : l'intervention est planifiée et apparaît dans votre calendrier
- **Négocier** : envoyez un contre-proposition ou des modifications
- **Refuser** : le devis est archivé, vous pouvez contacter un autre technicien

## Planification de l'intervention

Une fois le devis accepté :

1. Un **événement** est automatiquement créé dans le calendrier Hasfy
2. Un **ticket** peut être créé ou lié si ce n'est pas déjà fait
3. Le technicien reçoit une **confirmation** avec les détails de l'intervention
4. Des **rappels** sont envoyés avant l'intervention (24 h et 1 h avant)

### Suivi de l'intervention

Pendant et après l'intervention, le suivi se fait depuis le ticket associé :

- Le technicien peut ajouter des **notes** et des **photos** de l'intervention
- Le **temps passé** est enregistré pour la facturation
- Le statut du ticket évolue : En cours → Résolu
- Un **rapport d'intervention** est généré automatiquement

## Évaluations et avis

### Évaluer un technicien

Après chaque intervention terminée, vous êtes invité à évaluer le technicien :

1. **Note globale** : de 1 à 5 étoiles
2. **Critères détaillés** :
   - **Ponctualité** : respect des horaires convenus
   - **Compétence technique** : qualité de l'intervention
   - **Communication** : clarté des échanges et du rapport
   - **Rapport qualité/prix** : adéquation du tarif avec la prestation
3. **Commentaire** : texte libre décrivant votre expérience
4. **Recommandation** : recommanderiez-vous ce technicien ? (Oui/Non)

> **Note** : Les évaluations sont publiées après modération et restent anonymes (seul le nom de l'organisation est affiché, pas celui de l'évaluateur).

### Consulter les avis

Sur le profil de chaque technicien, les avis sont affichés par ordre chronologique inverse. Vous pouvez filtrer par note et par type d'intervention.

## Devenir technicien sur la marketplace

Si vous êtes technicien indépendant ou prestataire IT, vous pouvez rejoindre la marketplace Hasfy :

1. Créez un compte Hasfy avec le rôle **Technicien**
2. Accédez à **Marketplace &gt; Mon profil technicien**
3. Complétez votre profil : spécialités, certifications, zone d'intervention, tarifs
4. Soumettez votre candidature pour vérification
5. Après validation par l'équipe Hasfy (sous 48-72 h), votre profil est visible sur la marketplace

### Exigences

- Justificatif d'identité
- Preuve d'activité professionnelle (SIRET, Kbis ou équivalent)
- Au moins une certification IT reconnue *(recommandé mais non obligatoire)*
- Assurance responsabilité civile professionnelle

## Facturation

La facturation des interventions marketplace est gérée via l'intégration Stripe de Hasfy :

- Le paiement est sécurisé et séquestré jusqu'à la validation de l'intervention
- La facture est émise automatiquement après validation
- Les factures sont accessibles depuis **Paramètres &gt; Facturation &gt; Historique**
- Les techniciens reçoivent leur paiement sous 7 jours ouvrés après validation

Pour plus de détails sur la facturation, consultez la page [Intégration Stripe](/integrations/stripe).
