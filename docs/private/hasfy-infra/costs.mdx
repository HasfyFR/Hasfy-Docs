## 📉 Optimisations Possibles

### Phase 1 (Alpha) - Économies potentielles

| Optimisation | Économie |
|--------------|----------|
| Committed Use Discounts (1 an) | -20% |
| Spot VMs pour dev/test | -60-80% |
| **Potentiel** | **~50-100€/mois** |

### Général

| Optimisation | Impact |
|--------------|--------|
| ILM (Index Lifecycle Management) | Réduction stockage 30-50% |
| Snapshot vers GCS Nearline | Réduction coût backups 60% |
| Compression des indices | Réduction stockage 20-30% |
