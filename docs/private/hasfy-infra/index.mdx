## ⚡ Démarrage Rapide

```bash
# 1. Cloner le repo
git clone https://github.com/HasfyFR/Hasfy-Infra.git
cd Hasfy-Infra/providers/gcp

# 2. Infrastructure Terraform
cd terraform
terraform init && terraform apply

# 3. Déployer Elastic Stack
kubectl apply -f kubernetes/01-eck-operator/
kubectl apply -f kubernetes/02-elasticsearch/elasticsearch-phase1.yaml
kubectl apply -f kubernetes/03-kibana/kibana-phase1.yaml
kubectl apply -f kubernetes/04-fleet/fleet-server-phase1.yaml

# 4. Récupérer le mot de passe
kubectl get secret elasticsearch-es-elastic-user -n elastic-system \
  -o go-template='{{.data.elastic | base64decode}}'
```
